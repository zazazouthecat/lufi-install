# 🏁 Installation 🏁

### Télécharger directement depuis :

`wget https://git.io/JuvwC -O lufi-install.sh`

### Rendre le script executable:

`chmod +x lufi-install.sh`

### Exéctuer le script en root:

`./lufi-install.sh`
